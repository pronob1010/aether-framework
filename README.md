# Aether

A Python framework for building AI-native applications, blending classical
software engineering patterns with the demands of LLM-based execution.

**Status:** v0.1 in development.

## Design principles

1. Explicit over magical
2. Composable over monolithic
3. Observable by default
4. Failure-aware
5. Cost & latency aware
6. Typed contracts
7. Async-first
8. Testable without API calls

## Modules (current)

- `aether.llm` — Provider-agnostic LLM access (Strategy + Adapter)
